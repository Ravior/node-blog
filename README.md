node-blog
=========

在node.js平台上搭建的个人博客.  
express3+mysql构建!   

博客预览地址: http://supernode.duapp.com/   

安装
=======
1、进入应用目录 执行:
```
npm install 
````
自动会查找package.json中依赖的包并下载。

2、导入node_blog.sql数据库文件   
更改config.js中的数据库连接信息。


管理后台为: 访问路径+/admin   
用户名/密码 为admin/admin   可自由在config.js中更改
